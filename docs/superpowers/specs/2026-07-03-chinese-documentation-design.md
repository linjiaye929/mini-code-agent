# Mini CodeAgent 中文文档设计

## 目标

为中文学习者提供完整、可持续维护的项目入口，同时保留英文文档，兼顾国际开源读者。

## 范围

- 新增根目录 `README.zh-CN.md`，忠实覆盖当前英文 README 的 M8 功能、使用方法、安全边界和文档导航。
- 在英文 `README.md` 顶部增加语言切换链接。
- 为 `docs/architecture/` 下现有 16 篇核心架构文档增加同名 `.zh-CN.md` 中文版本。
- 中文 README 优先链接中文架构文档，现有中文学习与简历文档保持不变。
- 将 GitHub 仓库 Description 更新为简洁中文。

## 不在本次范围

- 不翻译 ADR、`docs/superpowers/plans/` 和历史规格文档。
- 不修改代码、配置、协议、测试或运行行为。
- 不改变英文文件的技术含义，也不把未实现功能写成已实现。

## 文件约定

- 英文原文：`name.md`
- 中文版本：`name.zh-CN.md`
- 两种语言在文档顶部互相链接。
- Python 标识符、命令、配置字段、协议名、错误码和代码块保持原样。

## 验收

- 中文 README 可以独立完成安装、配置、CLI/Web 启动和能力边界理解。
- 16 篇架构中文文档与英文文件一一对应，标题和章节结构一致。
- 所有本地 Markdown 链接存在。
- Markdown 格式检查、项目现有质量检查不因文档变更失败。
- GitHub 页面展示中文 Description，英文 README 仍可访问。
